# ESPNOW_Mesh_ControlAndMonitoring
IoT system using ESPNOW to form mesh network
- Webserver using Node Red run on AWS EC2 Ubuntu
- AWS IoT Core MQTT Broker
- Web dashboard: http://iotsystem.nhuthoang.online
